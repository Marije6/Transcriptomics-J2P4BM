# Data Stewardship (Beheren van data)

## Wat is Data Stewardship?
Data Stewardship betekent het zorgvuldig beheren, documenteren en structureren van data zodat onderzoek reproduceerbaar en controleerbaar is. Binnen dit project is gewerkt volgens de FAIR-principes: data moet vindbaar, toegankelijk, herbruikbaar en reproduceerbaar zijn.

---

## Belang van databeheer
Goed databeheer is belangrijk omdat het:
- fouten voorkomt
- analyse reproduceerbaar maakt
- samenwerking makkelijker maakt
- overzicht houdt in grote hoeveelheden data (zoals RNA-seq data)

---

## Structuur van data
De data in dit project is georganiseerd in vaste mappen:

- `data/ruw/` → originele sequencing data (niet aangepast)
- `data/processed/` → bewerkte data
- `scripts/` → R-scripts voor analyse
- `resultaten/` → figuren en output
- `bronnen/` → literatuur en referenties

Deze structuur zorgt ervoor dat de workflow logisch en reproduceerbaar is.

---

## Naamgeving en documentatie
Bestanden zijn duidelijk benoemd (bijv. `volcano_plot.png`, `DESeq2_results.csv`). Hierdoor is snel duidelijk wat elk bestand bevat.

Daarnaast is gebruikgemaakt van:
- README-bestand met uitleg
- commentaar in scripts

---

## Reproduceerbaarheid
Door het gebruik van scripts in R is de volledige analyse reproduceerbaar. Van ruwe data tot eindresultaten kan elke stap opnieuw uitgevoerd worden.
